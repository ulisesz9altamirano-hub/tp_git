💻 Descripción del Sistema

Este sistema es una plataforma web desarrollada para la gestión y administración de matrículas de estudiantes. Permite realizar operaciones CRUD completas (Alta, Baja, Modificación y Consulta) sobre un motor de base de datos relacional, además de incluir herramientas avanzadas de filtrado, búsqueda y ordenamiento dinámico.

🛠️ Tecnologías Utilizadas

  • Backend: Python con el micro-framework Flask para el enrutamiento y la lógica de negocio.
  • Base de Datos: PostgreSQL para la persistencia del modelo de datos de estudiantes.
  • Controladores / Drivers: Psycopg2 (binary) para la conexión y ejecución de consultas SQL seguras desde Python.
  • Contenedores: Docker para el despliegue rápido y aislado de la infraestructura local de la base de datos.
  • Frontend: Plantillas dinámicas estructuradas con Jinja2, basadas en HTML5 estándar y estilizadas mediante CSS3 nativo adaptable (Responsive Design).
  • Variables de Entorno: python-dotenv para resguardar las credenciales sensibles de conexión de manera segura.
